# 2026 데이터마이닝 Term Project

## 주제
- **데이터 분석을 통한 자동차 제조 공정 생산성 개선방안 도출**

## 데이터 개요
### process.xlsx
자동차 생산 설비별 가동 및 생산 정보를 담은 데이터.

| 변수명 | 정의 | 유형 | 설명 |
| --- | --- | --- | --- |
| 설비 | 작업이 진행된 설비 | str | LINE_A, LINE_B, LINE_C |
| 근무조 | 교대근무에 따른 근무조 | str | 야간, 주간 |
| 모델 | 생산 모델 | str | 니로, 소나타, 아반떼, 아이오닉, 투싼, 플러그인투싼 |
| 생산수량 | 생산 수량 | int | 해당 근무조의 생산량 |
| 가동률 | 가동시간 / 조업시간 | float | 전체 조업시간 중 설비가 실제로 가동된 시간 비율 |
| 시간가동률 | 가동시간 / 부하시간 | float | 부하시간 대비 정지 없는 가동 비율 |
| 성능가동률 | (생산량×단위생산시간) / 가동시간 | float | 가동시간 중 설비 목표 성능 발휘 비율 |
| 불량수량 | 생산 수량 중 불량 수량 | int |  |
| 작업자_사번 | 작업자 사번 | int |  |
| 조업시간 | 시업~종업 사이 총 시간 | float | 휴식, 식사, 수주물량 부족 등 계획정지시간 포함 |
| 부하시간 | 조업시간 - 계획정지시간 | float | 설비가 정상 상태에서 가동되어야 할 시간 |
| 가동시간 | 부하시간 - 정지시간 | float | 실제 설비 가동 시간 (5분 이상 정지 제외) |

### worker.xlsx
작업자 인적사항을 담은 데이터.

| 변수명 | 정의 | 유형 | 설명 |
| --- | --- | --- | --- |
| 사번 | 작업자 사번 | int |  |
| 성명 | 작업자 성명 | str |  |
| 나이 | 작업자 나이 | int |  |
| 성별 | 작업자 성별 | str | 남, 여 |
| 최종학력 | 작업자 최종학력 | str | 고등학교, 전문대학, 대학 |
| 입사일자 | 작업자 입사일자 | date | 연-월-일 |

## 평가 기준
- **EDA 및 가설 도출 (개인, 10점)**
  - 데이터 전처리 및 가설 도출을 위해 다양한 EDA 기술 활용
  - 도출된 가설이 타당하며 데이터로 근거 제시
- **데이터 분석 (개인, 10점)**
  - 가설에 대한 논리적이고 타당한 분석 수행
  - 분석 결과의 실용적 가치 및 활용 가능성

## 참고 정의
- **조업시간**: 1일 근무시간을 기준으로 휴식, 식사, 수주물량 부족 등 계획정지시간을 포함한 총 근무 시간.
- **부하시간**: 조업시간에서 계획정지시간을 제외한, 설비가 정상 상태에서 가동되어야 할 시간.
- **가동시간**: 부하시간에서 정지시간을 제외한 실제 설비 가동 시간(정지시간은 5분 이상 설비 비가동 구간).
- **정지시간**: 설비 고장, 기종 변경, 자재 품절, 유실 기간 등 비계획적 설비 정지 시간.
